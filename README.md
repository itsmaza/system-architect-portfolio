## Getting Started

```bash
# Clone the repository
git clone https://github.com/devmazaharul/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
